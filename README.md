# CineQuery AI Backend

This is the FastAPI backend for the CineQuery AI application, deployed on Hugging Face Spaces. It uses a local SentenceTransformer model for query embeddings, a local Cross-Encoder model for re-ranking, and ChromaDB for vector retrieval.
